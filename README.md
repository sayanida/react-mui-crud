A simple User Management Grid application built with **React** and **MUI**.  
You can add, read, update, and delete users easily.

## Features
- View a list of users in a MUI Grid
- Open a user to read its property values
- Add a new user via a Dialog
- Edit an existing user via a Dialog
- Delete a user
- Sort columns in the Grid
- Responsive view

## User Data
Each user includes the following properties:
- First Name
- Last Name
- Age
